Detailed Description

The index page holds links to each contact record.

To make these easier to navigate, they are sorted alphabetically in ascending order by surname then forename. Anchor links are provided for each letter to aid with navigation.

Limitation: Anchors are generated for A-Z only and do not account for accents. E.g. "Álvarez" would be grouped at the end under "Ω".
